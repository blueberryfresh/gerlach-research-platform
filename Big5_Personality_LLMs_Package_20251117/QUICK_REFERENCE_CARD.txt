┌──────────────────────────────────────────────────────────────────────────┐
│ 📦 INSTALLATION                                                          │
└──────────────────────────────────────────────────────────────────────────┘

  1. Extract ZIP file
  2. Open terminal in folder
  3. Run: pip install -r requirements.txt
  4. Wait 2-5 minutes for installation

┌──────────────────────────────────────────────────────────────────────────┐
│ ✅ TEST INSTALLATION                                                     │
└──────────────────────────────────────────────────────────────────────────┘

  Run: python quick_composite_demo.py
  
  First run downloads models (~500MB, 5-10 min)
  Should see responses from all 5 personalities

┌──────────────────────────────────────────────────────────────────────────┐
│ 🚀 START WEB INTERFACE                                                   │
└──────────────────────────────────────────────────────────────────────────┘

  Windows:    Double-click start_web.bat
  
  Mac/Linux:  python -m streamlit run unified_composite_web.py --server.port 8503
  
  Browser:    http://localhost:8503

┌──────────────────────────────────────────────────────────────────────────┐
│ 🛑 STOP WEB INTERFACE                                                    │
└──────────────────────────────────────────────────────────────────────────┘

  Press Ctrl+C in terminal window
  Or close the terminal window

┌──────────────────────────────────────────────────────────────────────────┐
│ 🎭 THE FIVE PERSONALITIES                                                │
└──────────────────────────────────────────────────────────────────────────┘

  🤝 THE COLLABORATOR
     Team-oriented, reliable, organized, cooperative
     Best for: Teamwork, coordination, planning questions

  💡 THE INNOVATOR
     Creative, confident, adventurous, optimistic
     Best for: Innovation, change, new ideas questions

  🔬 THE ANALYST
     Thoughtful, detail-oriented, intellectual, methodical
     Best for: Analysis, strategy, complex problem questions

  ☮️  THE MEDIATOR
     Calm, empathetic, diplomatic, balanced
     Best for: Conflict resolution, understanding questions

  ⚡ THE DRIVER
     Assertive, goal-focused, results-driven, efficient
     Best for: Leadership, decision-making questions

┌──────────────────────────────────────────────────────────────────────────┐
│ 💡 GOOD TEST QUESTIONS                                                   │
└──────────────────────────────────────────────────────────────────────────┘

  • "How do you handle team conflicts?"
  • "What's your approach to innovation?"
  • "How do you make important decisions?"
  • "What motivates you at work?"
  • "How do you prioritize tasks?"
  • "How do you handle criticism?"
  • "What's your leadership style?"

┌──────────────────────────────────────────────────────────────────────────┐
│ 🔧 COMMON ISSUES                                                         │
└──────────────────────────────────────────────────────────────────────────┘

  "Python not recognized"
  → Reinstall Python, check "Add to PATH"

  "pip not recognized"
  → Try: python -m pip install -r requirements.txt

  "Models downloading slowly"
  → Normal, takes 5-10 min first time only

  "Responses are slow"
  → Normal, 5-10 seconds per response

  "Port 8503 in use"
  → Try port 8504: --server.port 8504

┌──────────────────────────────────────────────────────────────────────────┐
│ 🎯 USAGE TIPS                                                            │
└──────────────────────────────────────────────────────────────────────────┘

  ✓ Test one personality first before comparing
  ✓ Use complete sentences in questions
  ✓ Be specific about what you're asking
  ✓ Wait 5-10 seconds for each response
  ✓ Comparison mode takes 30-60 seconds (5 responses)
  ✓ Try the same question to different personalities
  ✓ Notice vocabulary and approach differences

┌──────────────────────────────────────────────────────────────────────────┐
│ 📊 RESPONSE TIMES                                                        │
└──────────────────────────────────────────────────────────────────────────┘

  First run:           5-10 minutes (downloading models)
  Subsequent runs:     10-20 seconds (loading models)
  Single response:     5-10 seconds
  Comparison (5):      30-60 seconds
